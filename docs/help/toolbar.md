# The toolbar

The toolbar provides buttons for common actions in the proposed
workflow.

---

![Indietro](images/goBack.png) Go back in the browsing history. Same as
`Alt`-`Left`.

![Avanti](images/goForward.png) Go forward in the browsing history.
Same as `Alt`-`Right`.

---

![Mostra libro](images/viewBook.png) Go to the "Libro" branch and expand
it. Same as **Visualizza > Mostra libro**.

![Mostra personaggi](images/viewCharacters.png) Go to the "Personaggi"
branch and expand it. Same as **Visualizza > Mostra personaggi**.

![Mostra luoghi](images/viewLocations.png) Go to the "Luoghi"
branch and expand it. Same as **Visualizza > Mostra luoghi**.

![Mostra oggetti](images/viewItems.png) Go to the "Oggetti" branch and
expand it. Same as **Visualizza > Mostra oggetti**.

![Mostra linee narrative](images/viewArcs.png) Go to the "Linee narrative" branch
and expand it. Same as **Visualizza > Mostra linee narrative**.

![Mostra note di progetto](images/viewProjectnotes.png) Go to the "Note di progetto" 
branch and expand it. Same as **Visualizza > Mostra note di progetto**.

---

![Salva](images/save.png) Save the project. Same as **File > Salva** or
`Ctrl`-`S`.

![Blocca/Sblocca](images/lock.png) Toggle the lock status of the project.

![Aggiorna da manoscritto](images/updateFromManuscript.png) Import the
current manuscript. Same as selecting the manuscript under **Importa**.

![Export manuscript](images/manuscript.png) Export the manuscript for
editing. Same as **Esporta > Manoscritto per la modifica**, but without
confirmation for opening the document.

---

![Aggiungi](images/add.png) Add element. Same as `Ctrl`-`N`.

![Aggiungi figlio](images/addChild.png) Add child element. Same as
`Ctrl`-`Alt`-`N`.

![Aggiungi genitore](images/addParent.png) Add element on the parent's level.
Same as `Ctrl`-`Alt`-`Maiusc`-`N`.

![Elimina](images/remove.png) Delete selected elements. Same as `Canc`.

---

![Taglia](images/cut.png) Move the selected element from the tree to the
clipboard. Same as `Ctrl`-`X`.

![Copia](images/copy.png) Copy the selected element to the clipboard.
Same as `Ctrl`-`C`.

![Incolla](images/paste.png) Paste the element stored in the clipboard to
the tree. Same as `Ctrl`-`V`.

You can copy and paste the following tree elements via the system
clipboard:

-   Parts and chapters,
-   sections,
-   stages,
-   plot lines,
-   plot points,
-   characters,
-   locations,
-   items,
-   project notes.

> **Hint**
> 
> 
> If multiple elements are selected, only the first one will be copied.
> However, if the element has "children", these will also be copied and
> pasted.


> **Attention**
>
>
> Relationships are not included when copying to the clipboard. This also
> applies to the section viewpoint and for plot lines/plot points.

---

![Evidenzia elementi etichettati](images/tag.png) 
Highlight elements with a specified tag. This opens a tag selection dialog.
Same as **Visualizza > Evidenzia elementi etichettati**.

![Seleziona elementi evidenziati](images/select.png) 
Select all highlighted elements in the project tree for further action.

![Annulla evidenziazione](images/reset_highlight.png) Reset the highlighting of tree elements.
Same as **Visualizza > Annulla evidenziazione**.

While tree elements are highlighted, the criterion is displayed on the right of the reset button.

---

![Mostra/Nascondi visualizzatore testo](images/viewer.png) Toggle Text viewer. Same as
**Visualizza > Mostra/Nascondi visualizzatore testo** or `Ctrl`-`T`.

![Mostra/Nascondi proprietà](images/properties.png) Toggle Properties. Same as
**Visualizza > Mostra/Nascondi proprietà** or `Ctrl`-`Alt`-`T`.
